Zanata is for everyone

Zanata is written in Java and uses modern web technologies like JBoss, Seam, GWT, Hibernate, and a REST API. It currently suports translation of DocBook/Publican documentation through PO files. Projects can be uploaded to and downloaded from a Zanata server using a Maven plugin or a Python client.


The first release has been deployed at translate.zanata.org and is ready to accept documentation from jboss.org projects for translation.


Zanata is free software, licensed under the LGPL.

FAQ - Frequently Asked Questions

Q: How do I start to translate?

You need a Fedora/JBoss/other account (please follow instructions here). Once you have joined your language team and you're approved as a translator, go to the "Projects" tab to choose the project you would like to translate. Click on "translate" and away you go!

Q: Do I need to commit my translation when I'm done?

No! Every time you translate an entry your changes will be saved automatically when you're done editing. No need to commit anything.

Q: Does my translation need to get reviewed or approved?

No, there is no formal review process. When you finish a translation it is already considered approved. If you are unsure about your translation maybe get another member of your language team to double-check your work.

Q: Can I download .po-files to translate offline?

Not at this point in time. To make full use of Zanatas features, like the translation memory and the simultaneous editing, the translation should ideally be done online via the Zanata tool.

Q: How do I lock the file that I'm currently working on?

There is no need to! Several translators can work on the same file concurrently without interfering with each other. You will even see the changes other translators make almost in real-time on your screen.

Q: What do the different colours and fonts of the strings mean?

"Click here to start translating" Untranslated.
Text in italics (yellow border) The translation is considered "fuzzy", it needs review.
Ordinary text (green border) Translated text.
