Zanata is for everyone

Zanata is written in Java and uses modern web technologies like JBoss, Seam, GWT, Hibernate, and a REST API. It currently suports translation of DocBook/Publican documentation through PO files. Projects can be uploaded to and downloaded from a Zanata server using a Maven plugin or a Python client.


A public instance of Zanata is hosted at translate.zanata.org and is ready to accept content from open source projects for translation.


For projects hosted at jboss.org, please use the JBoss instance of Zanata at translate.jboss.org.


Zanata is free software, licensed under the LGPL.
